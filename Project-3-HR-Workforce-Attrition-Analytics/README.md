# HR Workforce & Attrition Analytics

## Project Overview

This project analyzes employee attrition patterns to identify workforce turnover drivers and retention risks using Python and SQL.

The project focuses on understanding how overtime, salary, experience, department, and job roles impact employee attrition.

## Tools Used

* Python
* SQL
* Pandas
* NumPy
* Matplotlib

## Key Findings

* Overall Attrition Rate: 16.12%
* Overtime employees showed significantly higher attrition.
* Sales department recorded the highest attrition rate.
* Laboratory Technicians and Sales Executives experienced the highest turnover.
* Lower-income employees showed higher attrition.
* Single employees experienced higher attrition compared to married employees.
* Less experienced employees were more likely to leave.

## Business Questions Solved

* What is the overall attrition rate?
* Which departments experience the highest attrition?
* Which job roles have the highest employee turnover?
* Does overtime influence attrition?
* Does salary impact retention?
* Does marital status affect attrition?
* How does experience influence attrition?
